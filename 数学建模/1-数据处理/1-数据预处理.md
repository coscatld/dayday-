## 人口数量年龄、经济产业情况等统计数据
- 对于定性数据，例如关于一群人的性别和文化程度；某些事件调查的满意程度，使用次数最多的值来不缺失
- 对于定量数据，例如关于一群人的身高和年龄等数据，使用整体的均值来补缺失

## 热力学温度、地形测量、地位等追求函数精准而不关心变化的数据
Newton 插值法适合已有若干个已知点，然后用这些点构造一个插值多项式来估计缺失点。 
它通常用于数据点之间存在明显函数关系的情况，比如某一时刻的温度、某一位置的高度、某一距离处的测量值。它不是简单地求平均，而是根据已知点推导出一条通过这些点的函数曲线，再用这个函数去求缺失值。

使用步骤是：1，确定自变量 $x$ 和因变量 $y$；2，列出已知点 $(x_0,y_0),(x_1,y_1),\cdots$；3，计算差商；4，构造 Newton 插值多项式；5，把缺失点对应的 $x$ 代入多项式求 $y$。 Newton 插值多项式一般写成：
$$
P_n(x)=f[x_0]+f[x_0,x_1](x-x_0)+f[x_0,x_1,x_2](x-x_0)(x-x_1)+\cdots
$$
其中 $f[x_0,x_1]$ 是一阶差商：
$$
f[x_0,x_1]=\frac{f(x_1)-f(x_0)}{x_1-x_0}
$$
例如已知某地温度数据为 $(1,10),(2,14),(4,22)$，现在要求 $x=3$ 时的温度，就可以用这三个点构造 Newton 插值多项式，然后代入 $x=3$。这种方法的特点是公式明确，适合点数不太多、数据比较稳定的情况。但是如果点很多，尤其是在区间边缘，可能出现震荡，也就是你笔记里写的龙格现象。

## 适用类型：零件加工，水库水流量，图像“基线漂移”，机器人轨迹等精度要求高、没有突变的数据

**样条插值法适合要求曲线自然、平滑、连续变化的数据。** 它不是用一个高次多项式穿过所有点，而是把整个区间切成很多小段，每一段用一个低次多项式，最常见的是**三次样条插值**。这样既能经过已知点，又能保证整体曲线比较平滑。

**使用步骤是：1，把数据点按照自变量 xxx 从小到大排列；2，确定每两个相邻点之间的小区间；3，每个小区间建立一个三次多项式；4，要求相邻区间在连接点处函数值、一阶导数、二阶导数连续；5，把缺失点所在区间找出来，代入该区间的三次多项式求值。** 三次样条每一段通常写成：
$$
S_i(x)=a_i+b_i(x-x_i)+c_i(x-x_i)^2 +d_i(x-x_i)^3
$$
它的优势是平滑性强，适合水库流量、机器人轨迹、图像灰度曲线、零件加工路径等数据。比如机器人运动轨迹不能突然改变方向，所以不能只要求点的位置准确，还要要求速度、加速度变化平滑，这种场景样条插值就比 Newton 插值更合适。

```python
import numpy as np
import pandas as pd
from scipy.interpolate import CubicSpline

# =========================
# 1. 统计数据：众数 / 均值填补
# =========================

df = pd.DataFrame({
    "gender": ["男", "女", None, "男", "男"],
    "age": [18, 19, np.nan, 21, 20]
})

# 定性数据：用众数填补
df["gender"] = df["gender"].fillna(df["gender"].mode()[0])

# 定量数据：用均值填补
df["age"] = df["age"].fillna(df["age"].mean())

print("统计数据填补结果：")
print(df)


# =========================
# 2. Newton 插值法
# =========================

def newton_interpolation(x_data, y_data, x):
    x_data = np.array(x_data, dtype=float)
    coef = np.array(y_data, dtype=float)
    n = len(x_data)

    # 计算差商表的第一行系数
    for j in range(1, n):
        coef[j:n] = (coef[j:n] - coef[j - 1:n - 1]) / (x_data[j:n] - x_data[0:n - j])

    # 计算 Newton 插值多项式的值
    result = coef[-1]
    for k in range(n - 2, -1, -1):
        result = result * (x - x_data[k]) + coef[k]

    return result

x_data = [1, 2, 4]
y_data = [10, 14, 22]

x_missing = 3
y_missing = newton_interpolation(x_data, y_data, x_missing)

print("Newton 插值结果：")
print(y_missing)


# =========================
# 3. 样条插值法
# =========================

x_data = np.array([1, 2, 4, 5], dtype=float)
y_data = np.array([10, 14, 22, 25], dtype=float)

spline = CubicSpline(x_data, y_data)

x_missing = 3
y_missing = spline(x_missing)

print("样条插值结果：")
print(y_missing)
```
